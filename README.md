# HabitFlow — CS50 Final Project

HabitFlow is a simple habit-tracking web application built with **Flask (Python)**, **HTML**, and **CSS** — no JavaScript.

## Features (upgraded)
- User registration & login (passwords hashed)
- Add / edit / delete habits
- Mark habits as done for today (one mark per day)
- View history for each habit
- Statistics page: last 7 days pattern + current streak
- Light / Dark mode toggle (no JavaScript — stored in session)
- English UI, CS50-friendly
